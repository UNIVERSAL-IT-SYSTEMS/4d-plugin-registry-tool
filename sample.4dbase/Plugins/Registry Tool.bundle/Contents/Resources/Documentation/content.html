Registry Tool

Plugin Identifier: 10016

Summary

This plugin allows the user to read and write to the registry on Windows.

Windows

RT Get binary value

success:=RT Get binary value (key; subkey; name; value)

key (LONGINT)

You can use a HKEY constant for key.

subkey (TEXT)

name (TEXT)

value (BLOB)

success (LONGINT)

Discussion

This command tries to read the specified registry key whose value of type REG_BINARY.

RT Set binary value

success:=RT Set binary value (key; subkey; name; value)

key (LONGINT)

You can use a HKEY constant for key.

subkey (TEXT)

name (TEXT)

value (BLOB)

success (LONGINT)

Discussion

This command tries to write the specified registry key whose value is of type REG_BINARY. A key will be created it it doesn't exist. As with all other commands, a value of 0 will be returned if the operation is successful. See documentation for a list of system error codes.
